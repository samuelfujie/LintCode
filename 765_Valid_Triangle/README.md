# 765. Valid Triangle
[ https://www.lintcode.com/problem/valid-triangle/description ]

Given three integers `a`, `b`, `c`, return `true` if they can form a triangle.

## Example
**Example 1:**

Input:
```sh
a = 2, b = 3, c = 4
```
Output:
```sh
true
```

**Example 2:**

Input:
```sh
a = 1, b = 2, c = 3
```
Output:
```sh
false
```
