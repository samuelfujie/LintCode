# 196. Missing Number
[ https://www.lintcode.com/problem/missing-number/description ]

Given an array contains N numbers of 0 .. N, find which number doesn't exist in the array.

## Example
**Example 1:**

Input:
```sh
[0,1,3]
```
Output:
```sh
2
```

**Example 2:**

Input:
```sh
[1,2,3]
```
Output:
```sh
0
```

## Challenge
Do it in-place with O(1) extra memory and O(n) time.
