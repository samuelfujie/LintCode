# 374. Spiral Matrix
[ https://www.lintcode.com/problem/spiral-matrix/description ]

Given a matrix of *m x n* elements (m rows, n columns), return all elements of the matrix in spiral order.

## Example
**Example 1:**

Input:
```sh
[[ 1, 2, 3 ], 
 [ 4, 5, 6 ], 
 [ 7, 8, 9 ]]
```
Output:
```sh
[1,2,3,6,9,8,7,4,5]
```

**Example 2:**

Input:
```sh
[[ 6,4,1 ], 
 [ 7,8,9 ]]
```
Output:
```sh
[6,4,1,9,8,7]
```
