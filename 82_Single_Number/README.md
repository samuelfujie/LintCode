# 82. Single Number
[ https://www.lintcode.com/problem/single-number/description ]

Given `2 * n + 1` numbers, every numbers occurs twice except one, find it.

## Example
**Example 1:**

Input:
```sh
[1,1,2,2,3,4,4]
```
Output:
```sh
3
```
Explanation:
Only 3 appears once

**Example 2:**

Input:
```sh
[0,0,1]
```
Output:
```sh
1
```
Explanation:
Only 1 appears once

## Challenge
One-pass, constant extra space.
