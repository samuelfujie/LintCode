# 544. Top k Largest Numbers
[ https://www.lintcode.com/problem/top-k-largest-numbers/description ]

Given an integer array, find the top k largest numbers in it.

## Example
**Example 1:**

Input:
```sh
[3, 10, 1000, -99, 4, 100]
k = 3
```
Output:
```sh
[1000, 100, 10]
```

**Example 2:**

Input:
```sh
[8, 7, 6, 5, 4, 3, 2, 1]
k = 5
```
Output:
```sh
[8, 7, 6, 5, 4]
```
