# 223. Palindrome Linked List
[ https://www.lintcode.com/problem/palindrome-linked-list/description ]

Implement a function to check if a linked list is a palindrome.

## Example
**Example 1:**

Input:
```sh
1->2->1
```
Output:
```sh
true
```

**Example 2:**

Input:
```sh
2->2->1
```
Output:
```sh
false
```

## Challenge
Could you do it in O(n) time and O(1) space?
