# 94. Binary Tree Maximum Path Sum
[ https://www.lintcode.com/problem/binary-tree-maximum-path-sum/description ]

Given a binary tree, find the maximum path sum.

The path may start and end at any node in the tree.

## Example
**Example 1:**

Input:
```sh
2
```
Output:
```sh
2
```

**Example 2:**

Input:
```sh
  1
 / \
2   3
```
Output:
```sh
6
```
