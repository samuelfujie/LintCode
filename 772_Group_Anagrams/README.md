# 772. Group Anagrams
[ https://www.lintcode.com/problem/group-anagrams/description ]

Given an array of strings, group anagrams together.

## Example
**Example 1:**

Input:
```sh
["eat","tea","tan","ate","nat","bat"]
```
Output:
```sh
[["ate","eat","tea"],
 ["bat"],
 ["nat","tan"]]
```

**Example 2:**

Input:
```sh
["eat","nowhere"]
```
Output:
```sh
[["eat"],
 ["nowhere"]]
```

## Notice
All inputs will be in lower-case.
